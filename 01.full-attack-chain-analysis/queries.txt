SPL Queries Used for Analysis

====================================================================================================================================================================================================================

1. DNS Analysis

Find top DNS activity

| stats count by src_ip
| sort -count

Find NXDOMAIN responses

| search response="NXDOMAIN"
| stats count by src_ip
| sort -count

Find suspicious domains

| stats count by domain,response
| sort -count

====================================================================================================================================================================================================================

2. SQL Injection Detection

Identify suspicious URLs

| stats count by uri
| sort -count

Investigate login endpoint

| search uri="*login.php*"
| stats count by uri

Identify attacker activity

| stats count by src_ip, uri
| sort -count

====================================================================================================================================================================================================================

3. File Upload Detection

Detect upload activity

| search uri="/upload.php"
| stats count by method,status

Identify upload source

| search uri="/upload.php"
| stats count by src_ip

====================================================================================================================================================================================================================

4. Webshell Detection

Detect webshell execution

| search uri="/uploads/shell.php"
| stats count by method,status

Identify source IP

| search uri="/uploads/shell.php"
| stats count by src_ip

====================================================================================================================================================================================================================

5. Attack Chain Correlation

View attack timeline

| table src_ip domain response uri method status

Top attack activity

| stats count by uri
| sort -count

Correlate attack activity

| stats count by src_ip, uri, method
| sort -count

Full investigation view

| table src_ip domain uri method status
